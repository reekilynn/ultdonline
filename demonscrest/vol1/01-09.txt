#　──こんなに大勢の注目を集めるのは、生まれて初めてかもしれない。
——大概从出生以来，还是第一次被这么多人关注。
#　ふとそんなことを考えてしまってから、ユウマはそんなわけないと否定した。
突然闪过的这一想法，被佑马立刻否定了。根本没有这回事。
#　雪（ゆき）花（はな）小（しよう）学（がつ）校（こう）に入学してから五年と二ヶ月が経（た）つ。夏休み明けには自由研究の発表をするし、帰りの会で委員会活動の報告をすることも日（にち）常（じよう）茶（さ）飯（はん）事（じ）だ。五年生の時には、《少年の主張》の作文がなぜかクラスの代表に選ばれて、全校生徒の前で音読する羽目に陥（おちい）ったことさえある。あの時に比べれば、いまユウマに視線を注いでいる生徒の数は約十分の一──であるはずなのに。
自从在雪花小学就读以来已经过了五年两个月。暑假结束的时候会进行自由研究的演讲，在每天结束的班会上报告委员会的活动也算是家常便饭。五年级的时候，不知为何自己写的作文《少年的主张》被选为班级代表，落入要对着全校学生朗读的状态。相比那个时候，如今将视线投向佑马的学生数量不过十分之一——本该是这样才对。
#　汗（あせ）で濡（ぬ）れた掌（てのひら）をズボンの側面に擦（こす）りつけながら、ユウマはサワとコンケンを除く二十人のクラスメイトを見回した。
佑马一边将满是汗水的手掌在裤子侧面擦来擦去，一边来回看向佐羽和近健以外的二十名同学。
#　リラックスしている生徒は一人もいない。皆（みな）の瞳（ひとみ）に浮（う）かぶのは、困（こん）惑（わく）、混乱、反発、不安、恐（きよう）怖（ふ）、焦（しよう）燥（そう）……そしてほんの少しの期待。すでに、ユウマたちが巨（きよ）大（だい）な怪（かい）物（ぶつ）を始末したという話が広まっていて、この状（じよう）況（きよう）を解決してくれるものと思っているのだろう。
没有一个同学放松下来。他们的眼中，浮现出困惑、混乱、排斥、不安、恐惧、焦躁……以及些微的期待。佑马等人干掉了巨大怪物这件事已经传开，同学们大概觉得状况已经得到解决了吧。
#　だが残念ながら、その期待には応えられない。ユウマはこれから、現在判明している事実──アルテアからは脱（だつ）出（しゆつ）不可能であり、恐（おそ）らく怪（かい）物（ぶつ）もあれ一匹だけではないということを皆（みな）に告げなくてはならないのだ。
然而遗憾的是，佑马无法回应他们的这一期待。接下来，佑马必须对大家告知目前判明的事实——不可能从Altair中脱离，怪物恐怕也不止一只。
#「……おい、もったいぶんなよ。なんか言うなら早くしろ」
「……喂，别在那浪费时间。有话赶紧说。」
#　イートインコーナーに陣（じん）取（ど）ったままの須（ス）鴨（ガモ）光（テル）輝（キ）が、苛（いら）立（だ）ちを隠（かく）さない声を出した。すかさず隣（となり）の三（ミ）園（ソノ）愛（ア）莉（リ）亜（ア）が「そうよ、みんなヒマじゃないんだからね！」と追（つい）随（ずい）する。木（キ）佐（サ）貫（ヌキ）櫂（カイ）は何も言わないが、長い前（まえ）髪（がみ）に隠（かく）れた両目からは感情を読み取れない。
仍然占据着餐饮角的须鸭光辉，以毫不掩饰的焦躁声音说着。紧接着，旁边的三园爱莉亚也说着「是啊，大家可没那么多工夫！」。木佐贯櫂什么都没有说，但佑马无法确定被长长的刘海遮住的双眼中的感情。

向那边看了一瞬之后，佑马缓缓吸了一口气，开始说道：
#「……みんなももう気付いてると思うけど、いまこのアルテアの中で、普（ふ）通（つう）じゃ考えられないことが起きてる」
「……我想大家应该也注意到了，如今这个Altair中，正发生着一些平常难以想象的事情。」
#　押し黙（だま）る二十人を順に見ながら、強（こわ）張（ば）る口を動かし続ける。
他向陷入沉默的二十人逐一看去，略显僵硬的嘴唇继续动着说道：
#「綿（ワタ）巻（マキ）さんが怪（かい）物（ぶつ）になったり、でっかい化け物がうろついてたり……さっきバリケードの前で暴れてたやつは僕たちが何とかしたけど、たぶん同じような奴（やつ）が他にもいるはずだ。そいつがまた襲（おそ）ってきたら、あのバリケードじゃ防げない」

#「簡単に言うけど、あんなクソでけぇ奴（やつ）をどうやって何とかしたんだよ!?」
「说得简单点吧，你们是怎么把那个大块头家伙搞定的！？」
#　スケボーコンビの穂（ホ）刈（カリ）が焦（じ）れた声を出し、瀬（セ）良（ラ）も何かを言おうとした。しかしそれより早く、新たな声が響（ひび）いた。
滑板二人组中的穗刈焦躁地问道，濑良也想说些什么。然而没等他开口，又有一个声音响起：
#「だったら、他の化け物が来る前にアルテアから出たほうがいいんじゃないの？」
「要是这样，在别的怪物过来之前从Altair出去不就行了吗？」
#　発言者は、長めの髪（かみ）を両（りよう）肩（かた）の上で二つ結びにして、黒（くろ）縁（ぶち）の眼鏡をかけた女子だった。
说话的人是将长发在肩膀位置扎成两束的，戴着黑框眼镜的女孩子。
#　出席番号六番、清（シ）水（ミズ）友（トモ）利（リ）。図書委員で、よく難しそうな本を読んでいる。視力はクレストのアイレンズで補正できるのに、敢（あ）えて眼鏡を使う理由は、訊（き）いても教えてくれなかった……と以前ナギが言っていた。
学号6号，清水友利（Shimizu Tomori）。身为图书委员的她，经常会阅读一些难懂的书籍。明明QLEST的镜片可以对视力进行补正，却刻意要戴眼镜的原因，就算问她她也不会回答……之前凪曾经这样说过。
#　友利の言葉に、数名の生徒が大きく頷（うなず）く。中にはいますぐここから出たいと言わんばかりに腰（こし）を浮（う）かせる者までいる。しかし友利はあくまで冷静に、落ち着いた声で意見を述べた。
好几个学生在听到友利的话后都点了点头，其中还有人带着现在就想出去的想法抬起了身体。然而友利却以冷静沉着的声音继续说道：
#「芦（アシ）原（ハラ）くんが他にも化け物がいるって考えた理由は、たぶん三階に行った藤（フジ）川（カワ）さんたちが一階に降りてこないからでしょ？　私も心配だけど、子供だけでどうにかしようとするのは無茶だわ。外に出て、大人の助けを呼ぶべきだと思う」
「芦原同学认为还有其他怪物存在的理由，大概是因为去了三楼的藤川同学她们还没有回到一楼对吧？虽然我也很担心她们，但如果只有小孩子的话，就算想做些什么也太过乱来了。我想还是应该到外面去寻求大人们的帮助。」
#　日（ひ）頃（ごろ）は物静かで、休み時間でもほとんど喋（しやべ）らない友利の理路整然とした主張に、仕切り屋の須（ス）鴨（ガモ）でさえ反論しようとしなかった。本当に、そうできたらどんなにいいか、とユウマ自身も考えながら口を開く。
面对平常十分安静，即使在休息时间也不怎么说话的友利条理清晰的主张，就算是身为分配者的须鸭也没有反驳。佑马一边想着「如果能做到这一点的话该多好啊」，一边开口回答：
#「残念だけど、それは無理だ。さっきトンガリ頭の怪（かい）物（ぶつ）を倒（たお）してから、エントランスの様子を確かめてきたんだ。自動ドアは開かないし、ガラスは真っ黒になってて、割ろうとしてもビクともしなかった」
「很遗憾，这一点我们做不到。刚才在打倒那个锥子头怪物之后，我们去确认过入口的情况了。自动门打不开，玻璃也是全黑的。虽然想要砸碎玻璃，但是一点反应都没有。」
#「…………」
「…………」
#　友利がレンズの奥（おく）で両目を細め、他の生徒たちも不安そうにざわめく。そんな中、挑（ちよう）発（はつ）的（てき）な声を出したのはやはり須鴨だった。
友利眯起了眼镜后面的双眼，其他的学生们也发出了略显不安的嘈杂声音。随后，须鸭不出意料地以挑衅一般的声音说道：
#「そりゃ、お前みてぇなチビじゃ自動ドアの強化ガラスを割るのは無理だろ。オレか大（オオ）野（ノ）なら、一発で……」
「那是因为，你这样的小不点根本砸不碎自动门的强化玻璃吧。换成我或者大野，一下子就能……」
#「僕じゃない。コンケンが、このハンマーでぶっ叩（たた）いてもヒビさえ入らなかった」
「不是我。近健用这个锤子砸过，连一道裂缝都没砸出来。」
#　ユウマが視線を向けると、一歩前に進み出たコンケンが、両手でブルージング・ハンマーを持ち上げてみせた。これがトンガリ頭からドロップした正（しよう）真（しん）正（しよう）銘（めい）の《武器》だとは誰（だれ）も思わないだろうが、大きさや重量感は充（じゆう）分（ぶん）に伝わるはずだ。
看到佑马转过来的视线，近健向前走了一步，用双手拿起了Bruising Hammer。虽然不会有什么人觉得这是锥子头怪物掉落的名副其实的《武器》，但无疑他们都能看出其具有足够的尺寸感和重量感。
#　先ほどハンマーを持ち上げようとして一センチも浮（う）かせられなかった大野が、横目で須（ス）鴨（ガモ）を見てから言った。
刚才想要拿起锤子却失败了的大野，侧过脸看着须鸭说道：
#「俺は信じるぜ。だいたい、自動ドアから外に出られるなら、逆に大人が……警備員とか警察とかがアルテアに入ってきてるはずだろ」
「我相信他。毕竟，既然能从自动门走到外面的话，反过来大人们……保安和警察应该也能进入Altair吧。」

#　即（そく）時（じ）の脱（だつ）出（しゆつ）を主張していた清（シ）水（ミズ）友（トモ）利（リ）があっさり頷（うなず）いたので、ユウマはいささか驚（おどろ）いた。この頭の柔（やわ）らかさを須鴨にも見習ってほしいと思いながら、説明を再開する。
主张立刻脱离的清水友利一下子点头同意，让佑马略显吃惊。他一边想着该让须鸭学习一下这种柔软的思维模式，一边继续解释了下去：

「大概，无法破坏的还不止是自动门。大厅的玻璃都成了无法看到外面的一片漆黑，QLEST也连不到网络。我想在找出这一异常情况的原因之前，都无法从Altair中脱离。」

当佑马终于说出这句话的时候，友利的嘴唇抿紧了，须鸭的面容也发生了扭曲，其他的同学们都露出了惊愕的表情。
#　脱（だつ）出（しゆつ）不可能。
无法脱离。
#　たくさんのマンガやゲームで使われてきた言葉だが、それが現実になってしまうと容易には受け入れがたい。黒く染まったガラスを自分の手で触（さわ）り、ハンマーで殴（なぐ）ってもびくともしないところを目（もく）撃（げき）したユウマでさえそうなのだから、クラスメイトたちの困（こん）惑（わく）は言葉だけでは解消されないだろう。
虽然这个词在众多漫画和游戏中都有出现，但一旦化为现实，还是很难让人轻易接受。连佑马自己在亲手触碰染成一片漆黑的玻璃，看到即使用锤子砸过去也没有任何反应的景象后都是那个样子，如今他面前的同学们，恐怕也无法只靠听着佑马的话就能从困惑中走出来吧。
#「……どうしても信じられないって人がいたら、あとでエントランスを見に行く時間を作るよ。でもいまは、この貴重な避（ひ）難（なん）場所……シェルターを守ることと、そのための《力》をみんなに修得してもらうことを最優先にしたい」
「……如果还有人实在无法相信的话，之后我们会给各位时间去入口处看一下。不过现在，我们需要保护这个珍贵的避难所……或者说掩体，为此需要优先让大家获得《力量》。」

终于说到目前状况的核心内容的佑马做了一个深呼吸。然而这时，须鸭光辉再一次出声打断。故意咂了下舌头的他，带着毫不掩饰的敌意说道：
#「チッ、もったいぶるんじゃねーよ！　そうやって色々回りくどいことを言って仕切るつもりだろうけど、お前らは最後にここに来たんだからな。最初にこのショッピングエリアを見つけてシェルターにしたのは学級委員のオレなんだから、これからどうするか決めるのも当然オレ……」
「切，别在这浪费时间了！本来我打算等你拐弯抹角地讲完再说的，可你们是最后才到这里的人吧！一开始发现这个购物区并建造掩体的人可是身为学级委员的我，接下来该决定怎么办的人毫无疑问也该是我……」
#　須鴨の台詞（せりふ）が中断したのは、ユウマが勢いよく右手を持ち上げたからだった。
佑马一下子抬起右手，打断了须鸭的台词。
#　ショッピングエリアの真ん中に立っているユウマから、イートインコーナーに陣（じん）取（ど）っている須（ス）鴨（ガモ）までは五メートルほども離（はな）れている。それでも須鴨は、自分に向けられたユウマの右手に何かを感じたかのように、少しだけ上体を引いた。
从站在购物区正中央的佑马，到占据了餐饮角的须鸭大约有五米左右的距离。即使如此，须鸭仿佛也从指向自己的佑马的右手上感觉到了什么一般，稍微抬起了上半身。

#「ヴェンタス（風よ）！」

#　広げた手の先にライトグリーンの光球が出現し、薄（うす）暗（ぐら）いショッピングエリアを照らし出した。須鴨が再び派手に仰（の）け反（ぞ）り、バランスを崩（くず）して椅（い）子（す）ごと床（ゆか）に転がったが、クラスメイトたちは誰（だれ）もそちらを見ようとしなかった。
在佑马伸出的手前，出现了淡绿色的光球，照亮了昏暗的购物区。须鸭又一次夸张地仰向后面，身体失去平衡，连同椅子一起倒在了地上，不过班上已经没有一个人看向他了。

所有人都睁大双眼，口中也漏出了喘息声。虽然这样已经有足够的说服力，但为了不让人指责自己使用了类似LED灯一样的机关，佑马还是决定继续下去。
#「ア（鳥）ヴ（と）ィ（な）ス（り）！」

#　形態詞を詠（えい）唱（しよう）すると、光球は渦（うず）巻（ま）く気流となり、小さな鳥を形作った。風属性の汎（はん）用（よう）魔（ま）法（ほう）、《風の小鳥（ウインド・バード）》。攻（こう）撃（げき）力（りよく）はないに等しいが、鳥をぶつけてモンスターの目を眩（くら）ませたり、離（はな）れたところにある物を落としたりできる。
咏唱出形态词后，光球化为旋转的气流，变成了小鸟的形状。这是风属性的通用魔法，《风之小鸟（Wind Bird）》。虽然攻击力基本没有，但被鸟撞到的怪物会发生目眩，也可以击落远处的东西。
#　ユウマは室内を見回し、壁（かべ）際（ぎわ）に動かされた陳（ちん）列（れつ）棚（だな）の一つに狙（ねら）いをつけた。
