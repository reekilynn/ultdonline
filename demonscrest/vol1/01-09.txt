* ## 9

#　──こんなに大勢の注目を集めるのは、生まれて初めてかもしれない。

#　ふとそんなことを考えてしまってから、ユウマはそんなわけないと否定した。

#　雪（ゆき）花（はな）小（しよう）学（がつ）校（こう）に入学してから五年と二ヶ月が経（た）つ。夏休み明けには自由研究の発表をするし、帰りの会で委員会活動の報告をすることも日（にち）常（じよう）茶（さ）飯（はん）事（じ）だ。五年生の時には、《少年の主張》の作文がなぜかクラスの代表に選ばれて、全校生徒の前で音読する羽目に陥（おちい）ったことさえある。あの時に比べれば、いまユウマに視線を注いでいる生徒の数は約十分の一──であるはずなのに。

#　汗（あせ）で濡（ぬ）れた掌（てのひら）をズボンの側面に擦（こす）りつけながら、ユウマはサワとコンケンを除く二十人のクラスメイトを見回した。

#　リラックスしている生徒は一人もいない。皆（みな）の瞳（ひとみ）に浮（う）かぶのは、困（こん）惑（わく）、混乱、反発、不安、恐（きよう）怖（ふ）、焦（しよう）燥（そう）……そしてほんの少しの期待。すでに、ユウマたちが巨（きよ）大（だい）な怪（かい）物（ぶつ）を始末したという話が広まっていて、この状（じよう）況（きよう）を解決してくれるものと思っているのだろう。

#　だが残念ながら、その期待には応えられない。ユウマはこれから、現在判明している事実──アルテアからは脱（だつ）出（しゆつ）不可能であり、恐（おそ）らく怪（かい）物（ぶつ）もあれ一匹だけではないということを皆（みな）に告げなくてはならないのだ。

#「……おい、もったいぶんなよ。なんか言うなら早くしろ」

#　イートインコーナーに陣（じん）取（ど）ったままの須（ス）鴨（ガモ）光（テル）輝（キ）が、苛（いら）立（だ）ちを隠（かく）さない声を出した。すかさず隣（となり）の三（ミ）園（ソノ）愛（ア）莉（リ）亜（ア）が「そうよ、みんなヒマじゃないんだからね！」と追（つい）随（ずい）する。木（キ）佐（サ）貫（ヌキ）櫂（カイ）は何も言わないが、長い前（まえ）髪（がみ）に隠（かく）れた両目からは感情を読み取れない。

#　そちらを一（いつ）瞬（しゆん）だけ見てから、ユウマはゆっくりと息を吸い、話し始めた。

#「……みんなももう気付いてると思うけど、いまこのアルテアの中で、普（ふ）通（つう）じゃ考えられないことが起きてる」

#　押し黙（だま）る二十人を順に見ながら、強（こわ）張（ば）る口を動かし続ける。

#「綿（ワタ）巻（マキ）さんが怪（かい）物（ぶつ）になったり、でっかい化け物がうろついてたり……さっきバリケードの前で暴れてたやつは僕たちが何とかしたけど、たぶん同じような奴（やつ）が他にもいるはずだ。そいつがまた襲（おそ）ってきたら、あのバリケードじゃ防げない」

#「簡単に言うけど、あんなクソでけぇ奴（やつ）をどうやって何とかしたんだよ!?」

#　スケボーコンビの穂（ホ）刈（カリ）が焦（じ）れた声を出し、瀬（セ）良（ラ）も何かを言おうとした。しかしそれより早く、新たな声が響（ひび）いた。

#「だったら、他の化け物が来る前にアルテアから出たほうがいいんじゃないの？」

#　発言者は、長めの髪（かみ）を両（りよう）肩（かた）の上で二つ結びにして、黒（くろ）縁（ぶち）の眼鏡をかけた女子だった。

#　出席番号六番、清（シ）水（ミズ）友（トモ）利（リ）。図書委員で、よく難しそうな本を読んでいる。視力はクレストのアイレンズで補正できるのに、敢（あ）えて眼鏡を使う理由は、訊（き）いても教えてくれなかった……と以前ナギが言っていた。

#　友利の言葉に、数名の生徒が大きく頷（うなず）く。中にはいますぐここから出たいと言わんばかりに腰（こし）を浮（う）かせる者までいる。しかし友利はあくまで冷静に、落ち着いた声で意見を述べた。

#「芦（アシ）原（ハラ）くんが他にも化け物がいるって考えた理由は、たぶん三階に行った藤（フジ）川（カワ）さんたちが一階に降りてこないからでしょ？　私も心配だけど、子供だけでどうにかしようとするのは無茶だわ。外に出て、大人の助けを呼ぶべきだと思う」

#　日（ひ）頃（ごろ）は物静かで、休み時間でもほとんど喋（しやべ）らない友利の理路整然とした主張に、仕切り屋の須（ス）鴨（ガモ）でさえ反論しようとしなかった。本当に、そうできたらどんなにいいか、とユウマ自身も考えながら口を開く。

#「残念だけど、それは無理だ。さっきトンガリ頭の怪（かい）物（ぶつ）を倒（たお）してから、エントランスの様子を確かめてきたんだ。自動ドアは開かないし、ガラスは真っ黒になってて、割ろうとしてもビクともしなかった」

#「…………」

#　友利がレンズの奥（おく）で両目を細め、他の生徒たちも不安そうにざわめく。そんな中、挑（ちよう）発（はつ）的（てき）な声を出したのはやはり須鴨だった。

#「そりゃ、お前みてぇなチビじゃ自動ドアの強化ガラスを割るのは無理だろ。オレか大（オオ）野（ノ）なら、一発で……」

#「僕じゃない。コンケンが、このハンマーでぶっ叩（たた）いてもヒビさえ入らなかった」

#　ユウマが視線を向けると、一歩前に進み出たコンケンが、両手でブルージング・ハンマーを持ち上げてみせた。これがトンガリ頭からドロップした正（しよう）真（しん）正（しよう）銘（めい）の《武器》だとは誰（だれ）も思わないだろうが、大きさや重量感は充（じゆう）分（ぶん）に伝わるはずだ。

#　先ほどハンマーを持ち上げようとして一センチも浮（う）かせられなかった大野が、横目で須（ス）鴨（ガモ）を見てから言った。

#「俺は信じるぜ。だいたい、自動ドアから外に出られるなら、逆に大人が……警備員とか警察とかがアルテアに入ってきてるはずだろ」

#「…………それもそうね」

#　即（そく）時（じ）の脱（だつ）出（しゆつ）を主張していた清（シ）水（ミズ）友（トモ）利（リ）があっさり頷（うなず）いたので、ユウマはいささか驚（おどろ）いた。この頭の柔（やわ）らかさを須鴨にも見習ってほしいと思いながら、説明を再開する。

#「多分、壊（こわ）せないのは自動ドアだけじゃない。ロビーのガラスは全部真っ黒で、外はまったく見えないしクレストもネットに繫（つな）がらない。この異常の原因を突き止（と）めない限り、アルテアからは脱（だつ）出（しゆつ）不可能だと思う」

#　ユウマがついにその言葉を口にした途（と）端（たん）、友利は唇（くちびる）を引き結び、須鴨は大きく顔を歪（ゆが）め、他のクラスメイトたちも愕（がく）然（ぜん）とした表情を浮（う）かべた。

#　脱（だつ）出（しゆつ）不可能。

#　たくさんのマンガやゲームで使われてきた言葉だが、それが現実になってしまうと容易には受け入れがたい。黒く染まったガラスを自分の手で触（さわ）り、ハンマーで殴（なぐ）ってもびくともしないところを目（もく）撃（げき）したユウマでさえそうなのだから、クラスメイトたちの困（こん）惑（わく）は言葉だけでは解消されないだろう。

#「……どうしても信じられないって人がいたら、あとでエントランスを見に行く時間を作るよ。でもいまは、この貴重な避（ひ）難（なん）場所……シェルターを守ることと、そのための《力》をみんなに修得してもらうことを最優先にしたい」

#　いよいよ状（じよう）況（きよう）説明の核（かく）心（しん）に入るべく、ユウマは一度深呼吸した。しかしそこで再び須（ス）鴨（ガモ）光（テル）輝（キ）の横（よこ）槍（やり）が入った。わざとらしい舌打ちに続いて、敵意丸出しの言葉が飛んでくる。

#「チッ、もったいぶるんじゃねーよ！　そうやって色々回りくどいことを言って仕切るつもりだろうけど、お前らは最後にここに来たんだからな。最初にこのショッピングエリアを見つけてシェルターにしたのは学級委員のオレなんだから、これからどうするか決めるのも当然オレ……」

#　須鴨の台詞（せりふ）が中断したのは、ユウマが勢いよく右手を持ち上げたからだった。

#　ショッピングエリアの真ん中に立っているユウマから、イートインコーナーに陣（じん）取（ど）っている須（ス）鴨（ガモ）までは五メートルほども離（はな）れている。それでも須鴨は、自分に向けられたユウマの右手に何かを感じたかのように、少しだけ上体を引いた。

#　──案外、勘（かん）はいい奴（やつ）なのかもな。

#　そんなことを考えながら、ユウマは風の属性詞を唱えた。

#「ヴェンタス（風よ）！」

#　広げた手の先にライトグリーンの光球が出現し、薄（うす）暗（ぐら）いショッピングエリアを照らし出した。須鴨が再び派手に仰（の）け反（ぞ）り、バランスを崩（くず）して椅（い）子（す）ごと床（ゆか）に転がったが、クラスメイトたちは誰（だれ）もそちらを見ようとしなかった。

#　全員の両目がいっぱいに見開かれ、口からは喘（あえ）ぎ声（ごえ）が漏（も）れる。これだけでも充（じゆう）分（ぶん）な説得力があるだろうが、ＬＥＤライトか何かを使ったトリックだと言われないように、ダメを押しておくことにする。

#「ア（鳥）ヴ（と）ィ（な）ス（り）！」

#　形態詞を詠（えい）唱（しよう）すると、光球は渦（うず）巻（ま）く気流となり、小さな鳥を形作った。風属性の汎（はん）用（よう）魔（ま）法（ほう）、《風の小鳥（ウインド・バード）》。攻（こう）撃（げき）力（りよく）はないに等しいが、鳥をぶつけてモンスターの目を眩（くら）ませたり、離（はな）れたところにある物を落としたりできる。

#　ユウマは室内を見回し、壁（かべ）際（ぎわ）に動かされた陳（ちん）列（れつ）棚（だな）の一つに狙（ねら）いをつけた。

#「イグニス（飛べ）！」

#　発動詞を受け、緑色の小鳥は勢いよく羽ばたいて飛び立った。清（シ）水（ミズ）友（トモ）利（リ）の頭上を突っ切（き）り、陳（ちん）列（れつ）棚（だな）の最上部に命中してから、空気に溶（と）けるように消（しよう）滅（めつ）する。棚（たな）に載（の）っていたアルテアの形のミニクッションが一つ床（ゆか）に落ち、乾（かわ）いた音を立てる。

#　ユウマの実演が終わっても、二十人の生徒たちはしばらく黙（だま）り込んだままだった。やがて、あちこちで囁（ささや）き声（ごえ）が交（か）わされ始める。

#「いまの……魔（ま）法（ほう）だよな？　アクマジの……」

#「うん……あたし、ゲームの中で同じ呪（じゆ）文（もん）使ったもん……」

#「でも、ここは現実世界だろ……何かのトリックじゃねーのか……？」

#「それか、クレストのＡＲ映像かも……」

#　半信半疑の生徒たちを再び黙（だま）らせたのは、ユウマの正面に立つ清水友利のひと言だった。

#「いま、風を感じた」

#　右手の指先で自分の頰（ほお）を撫（な）でながら続ける。

#「クレストは単体じゃ触（しよつ）覚（かく）をサポートしてないから、空気が動いたならＡＲ映像じゃないわ。ＬＥＤライトとミニ扇（せん）風（ぷう）機（き）を使ったとしても、光を小鳥の形に変えて羽ばたかせたり、物を落としたりするのは無理よ。いまのはトリックじゃない……本物の魔（ま）法（ほう）だわ」

#　黒（くろ）縁（ぶち）眼鏡を光らせながら友利がそう言い切ると、もう反論しようとする生徒はいなかった。須（ス）鴨（ガモ）は尻（しり）餅（もち）を突いたままぽかんと口を開け、取り巻きの三（ミ）園（ソノ）愛（ア）莉（リ）亜（ア）と木（キ）佐（サ）貫（ヌキ）櫂（カイ）も何も言おうとしない。

#　次に声を上げたのは、バスケクラブの大（オオ）野（ノ）曜（ヨウ）一（イチ）だった。

#「……もしかして、魔（ま）法（ほう）を使ってあの化け物を倒（たお）したのか？　あいつだけじゃない……綿（ワタ）巻（マキ）も、魔（ま）法（ほう）で倒（たお）した……いや、殺したのかよ？」

#「違（ちが）う」

#　生徒たちが動（どう）揺（よう）する前に、ユウマは急いで否定した。

#「ここに押し入（い）ろうとしてた化け物は確かに殺したけど、綿巻さんは殺してない。魔（ま）法（ほう）で……拘（こう）束（そく）して、動けなくしたんだ。僕は、綿巻さんを元に戻（もど）す方法を探すつもりだ」

#　この段階で「捕獲（キヤプチヤー）魔（ま）法（ほう）でしてカード化した」とは言えず、ユウマは拘（こう）束（そく）という言葉を使ったが、幸いそこを追（つい）及（きゆう）しようとする生徒はいなかった。トンガリ頭ことコーンヘッド・ブルーザーも実際には魔（ま）法（ほう）ではなく消毒用エタノールを使ったのだが、それは追々説明すればいいだろう。いまは皆（みな）に、《ジョブチェンジ》の準備をさせるほうが先だ。

#　ユウマの宣言を聞いた大野は、表情を目まぐるしく変化させてから、どこか縋（すが）るような声を絞（しぼ）り出した。

#「……元に、戻（もど）す……？　戻（もど）せるのか、綿巻を？」

#「絶対に戻（もど）せるって約束はできない。けど、方法はあると思う。僕はそう信じてる」

#　それはユウマの噓（うそ）偽（いつわ）りない本心だった。大野にもそれが伝わったのだろう、細く、長く息を吐（は）いてからゆっくりと頷（うなず）いた。

#「……解（わか）った。芦（アシ）原（ハラ）、俺はお前を信じる。これからは、お前の指示に従う」

#「ありがとう、大（オオ）野（ノ）」

#　気（き）恥（は）ずかしさを押し殺（ころ）して礼を言ってから、他の生徒たちの様子を確かめる。穂（ホ）刈（カリ）と瀬（セ）良（ラ）、それに清（シ）水（ミズ）友（トモ）利（リ）にも異論はないようだ。言い方は悪いが、この四人さえ押さえられれば主導権は確保できたも同然だろう。友利は決して一組女子のリーダー格というわけではないものの、そのポジションにある藤（フジ）川（カワ）憐（レン）や寺（テラ）上（ガミ）京（キヨウ）香（カ）は三階に向かった十五人の中に含（ふく）まれていて、ここにはいない。

#　気がかりなのはやはり須（ス）鴨（ガモ）のグループだが、この極限状（じよう）況（きよう）で須鴨の独りよがりな言動は皆（みな）の反発を買っている気配があるし、ギャル系の三（ミ）園（ソノ）愛（ア）莉（リ）亜（ア）はヒエラルキー上位ではあっても女子の中では浮（う）き気味なようだ。

#　ユウマはクラス内の勢力争いに興味はないが、これから全員をジョブチェンジさせることを考えれば慎（しん）重（ちよう）にならざるを得ない。攻（こう）撃（げき）魔（ま）法（ほう）を操（あやつ）る力や人（にん）間（げん）離（ばな）れした腕（わん）力（りよく）を得た生徒たちが、それを好き放題に試（ため）したりすれば俄（にわか）作（づく）りのシェルターなどあっという間に崩（ほう）壊（かい）してしまう。この場の全員が自分の置かれた状（じよう）況（きよう）を認（にん）識（しき）し、受け入れ、落ち着きを取（と）り戻（もど）すまでは、ユウマとサワ、コンケンの三人で主導権を確保し続けるしかないのだ。

#　最後にもう一度クラスメイトたちの顔を見回し、ユウマは言った。

#「じゃあ、これから、みんなに魔（ま）法（ほう）の使い方を教える」

#　怪（け）我（が）をしてブランケットに横たわる多（タ）田（ダ）智（トモ）則（ノリ）や会（アイ）田（ダ）慎（シン）太（タ）を含（ふく）め、全員が食い入るような視線を注いでくる。耳が痛くなるような沈（ちん）黙（もく）の中、ユウマはふと理由のない不安を感じた。自分が何か、とても重要なことを見落としているという感覚。

#　だが、もうこれ以上は引き延ばせない。両（りよう）拳（こぶし）を強く握（にぎ）り、不安感を振り払（はら）ってから最後の注意事（じ）項（こう）を告げる。

#「僕たちは《ジョブチェンジ》って言ってるけど、それをすると左手のクレストが変形して、肘（ひじ）のあたりまで伸びてくる。その時に熱く感じるかもだけど火傷（やけど）はしないから我（が）慢（まん）してくれ。……じゃあ、みんな、仮想デスクトップにあるアクチュアル・マジックのアイコンを押して」

#　その指示を聞いた途（と）端（たん）、穂（ホ）刈（カリ）陽（ハル）樹（キ）が目を丸くする。

#「アクマジのアイコンって……あれ、カリキュラスの中じゃねーと起動しねーだろ」

#「いいから、押すだけ押してみてくれよ」

#　ユウマがそう言っても、生徒たちは不安そうに顔を見合わせている。須（ス）鴨（ガモ）は明らかに様子見の態勢だし、大（オオ）野（ノ）でさえなかなかきっかけが摑（つか）めないようだ。

#　最初に右手を動かしたのは、図書委員の清（シ）水（ミズ）友（トモ）利（リ）だった。ほっそりした人差し指を眼鏡の前まで持ち上げてから、左下に滑（すべ）らせる。そこで一（いつ）瞬（しゆん）指が止まったが、左手も持ち上げて右手に添（そ）え、ユウマには見えないアイコンを押し込む。

#　友（トモ）利（リ）の左手の甲（こう）に貼（は）られたクレストが、鮮（あざ）やかなエメラルドグリーンに輝（かがや）いた。口から、細い悲鳴が漏（も）れる。

#「……っ……あっ……」

#　それを見て、生徒たちはいっそう不安そうな顔になったが、何人かは負けん気を刺（し）激（げき）されたのか続いてアイコンを押した。その中には大（オオ）野（ノ）も含（ふく）まれていて、左手が赤い光に包まれた途（と）端（たん）に低い声で呻（うめ）いたが、すぐに大声で叫（さけ）ぶ。

#「だ……大（だい）丈（じよう）夫（ぶ）だ！　あっちぃけど、我（が）慢（まん）できないほどじゃねーぞ！」
